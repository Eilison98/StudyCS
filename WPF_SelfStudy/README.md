## ERP

</br>

- 구현기능
  - DB 재고실사현황 테이블 전체조회

- 현재 구현하지 못한( 추가할 )기능
  - DB 유저테이블 추가
  - DB 입고테이블 추가
  - DB 출고테이블 추가
  - DB 로그인정보 테이블 추가
  - 미구현 UI추가
